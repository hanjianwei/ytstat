# Ytstat

[![Gem Version](https://img.shields.io/gem/v/ytstat.svg)](https://rubygems.org/gems/ytstat)

Summary of [Yunti](http://ytvpn.com) servers.

## Installation

Add this line to your application's Gemfile:

    gem 'ytstat'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ytstat

## Usage

``` bash
ytstat
```
